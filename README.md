# 🔔 AI-Powered Reply Detection & Notification System
### n8n · OpenAI GPT-4 · Gmail · Google Sheets

![Notification Automation Workflow](notification_automation.png)

---

## 📌 Overview

An automated reply monitoring system that continuously scans active outreach threads, classifies incoming responses using AI, notifies the sender of real human replies in real time, and updates the CRM automatically — all without any manual checking.

Runs every 2 hours, processes all active leads in a single execution loop, and ensures no reply ever goes unnoticed.

---

## 🏗️ System Architecture

```
Schedule Trigger (every 2 hours)
        ↓
Get row(s) in sheet
[Filter: done=true AND Checked reply=false]
        ↓
Loop Over Items
[Processes all matching leads in one run]
        ↓
Get a thread (Gmail)
[Fetches full conversation thread]
        ↓
Code in JavaScript2
[AI-powered reply classification]
        ↓
Switch (Routes by result)
   ├── Result 1 (Real reply)   → Send notification + Update CRM
   ├── Result 2 (No reply)     → Update CRM silently
   └── Result 3 (Auto-reply)   → Update CRM silently
        ↓
Update row in sheet1
[Marks Checked reply = true]
        ↓
Loop continues to next lead
```

## ✨ Key Features

### 🔍 Bulk Thread Monitoring
- Fetches all leads where email was sent but reply not yet checked
- Processes every active lead in a single scheduled run
- Uses n8n's Loop Over Items for efficient batch processing
- Never misses a reply regardless of volume

### 🧠 AI Reply Classification
Analyzes each Gmail thread and classifies the response:

| Result | Classification | Action Taken |
|--------|---------------|--------------|
| `1` | Real human reply | Notify sender + update CRM |
| `2` | No reply | Update CRM silently |
| `3` | Auto-reply / OOO | Update CRM silently |

Detection engine uses:
- Gmail message header analysis (`Auto-Submitted`, `X-Autoreply`, `Precedence`)
- 15+ vacation/OOO keyword matching
- Thread message count logic
- `SENT` label filtering to skip outbound messages

### 📲 Instant Reply Notifications
When a real human reply is detected:
- Immediate Gmail notification sent to sender
- Notification includes prospect name, company and reply context
- CRM updated to prevent duplicate notifications
- Sequence automatically flagged for manual follow-up

### 📊 Automatic CRM Updates
Every checked thread updates Google Sheets:

| Field | Value Set |
|-------|----------|
| `Checked reply` | `true` |
| `issues` | Reply classification result |

## 📋 Node Breakdown

| Node | Function |
|------|----------|
| `Schedule Trigger` | Fires every 2 hours, 24/7 |
| `Get row(s) in sheet` | Pulls leads with `done=true` + `Checked reply=false` |
| `Loop Over Items` | Iterates through all matching leads |
| `Get a thread` | Fetches full Gmail thread by thread ID |
| `Code in JavaScript2` | Classifies reply as 1 / 2 / 3 |
| `Switch` | Routes based on classification result |
| `Send a message1` | Sends real-time reply notification (result=1 only) |
| `Update row in sheet1` | Marks thread as checked in CRM |
